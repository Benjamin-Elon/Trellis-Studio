# Trellis splash backgrounds

Place packaged splash backgrounds in this directory before building Trellis Studio.

`trellis-garden-sunrise.png` is the initial packaged scene. <!-- NEW -->

- Supported formats: `.webp`, `.jpg`, `.jpeg`, and `.png`.
- Recommended shape: landscape at roughly 16:9, using 1672x941 pixels or larger. <!-- CHANGE -->
- Keep important subjects away from the center, where the splash card appears.
- There is no enforced size limit; balance image fidelity against package size. <!-- CHANGE -->

The app chooses one file with pure randomness on each application launch. An empty
directory is valid and uses the built-in botanical gradient.
